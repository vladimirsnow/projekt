## 2. Что Это За Проект На Самом Деле

По коду проект представляет собой одностраничную платформу онлайн-коммуникации и обмена сообщениями в стиле Telegram.

Что реализовано:

- регистрация и вход по нику, email и паролю
- восстановление пароля по email
- сохранение сессии через токен в localStorage
- личные чаты
- групповые чаты
- публичные и приватные каналы
- поиск пользователей по нику
- поиск чатов и поиск каналов для подписки
- отправка текстовых сообщений в реальном времени
- отправка файлов и медиа
- предпросмотр файлов перед отправкой
- просмотр изображений и видео в отдельном окне
- реакции на сообщения
- индикатор набора текста
- управление участниками и ролями администраторов
- смена ника, email, пароля и аватара
- аудио- и видеозвонки через WebRTC
- адаптивный интерфейс для мобильных устройств

Основные файлы проекта:

- `public/index.html` - структура интерфейса
- `public/styles/main.css` - стили, сетка, адаптивность
- `public/scripts/main.js` - клиентская логика
- `server.js` - сервер, Socket.IO, API, загрузка файлов, сессии, восстановление пароля
- `data/users.json` - пользователи
- `data/chats.json` - чаты и сообщения
- `uploads/` - загруженные файлы

### 18.8 2.8 Производительность И Ограничения

С точки зрения производительности проект использует хранение данных в памяти с последующей синхронизацией в JSON-файлы.
